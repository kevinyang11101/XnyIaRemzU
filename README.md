# 酒店推荐管理系统 java196

## 项目概述

酒店推荐管理系统是一个集前台展示页面和后台管理功能于一体的平台。它采用了协同过滤算法，为用户提供酒店推荐服务，同时支持管理员和用户两种角色，以满足不同的业务需求。

## 技术栈

- **后端技术**：Spring Boot，MyBatis
- **前端技术**：Vue，ElementUI，Axios，ECharts
- **系统环境**：IDEA，JDK 1.8，MySQL，Maven

## 功能模块

### 前台功能

- 首页推荐信息
- 系统简介
- 关于我们
- 公告资讯
- 酒店客房展示
- 公共设施展示
- 周边环境展示
  - 浏览、收藏、点赞、点踩、预订、评论

### 管理后台功能

- 数据报表
- 员工管理
- 用户管理
- 房间类型管理
- 酒店客房管理
- 酒店费用报表
- 类型统计
- 入住与退房登记
- 公共设施管理
- 周边环境管理
- 预定取消审核
- 系统管理
  - 系统简介
  - 轮播图管理
  - 公告资讯管理
  - 在线资讯

## 系统角色

- 管理员
- 用户

## 运行环境

系统使用以下环境可以直接运行启动：

- IDEA
- JDK 1.8
- MySQL
- Maven

## 目录结构

```plaintext
/hotel-recommendation-system
|-- /backend                # 后端代码目录
|-- /frontend               # 前端代码目录
|-- /sql                    # 数据库脚本目录
|-- pom.xml                 # Maven项目配置文件
```

- 集成协同过滤算法，为用户提供精准的酒店推荐服务
- 采用前后端分离的技术架构，便于维护与扩展
- 功能模块齐全，覆盖酒店管理的多个方面
